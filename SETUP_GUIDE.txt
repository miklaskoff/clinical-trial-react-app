STEP-BY-STEP SETUP (5 MINUTES)
================================================================================

STEP 1: CHECK IF YOU HAVE NODE.JS
----------------------------------
Open terminal/command prompt and type:
   node --version

If you see a version number (like v18.0.0), you're good!
If not, go to https://nodejs.org/ and install it.


STEP 2: CREATE PROJECT FOLDER
----------------------------------
1. Create a new folder called "clinical-trial-app"
2. Put ALL downloaded files in this folder


STEP 3: ORGANIZE FILES
----------------------------------

STEP 4: INSTALL DEPENDENCIES
----------------------------------
1. Open terminal in the clinical-trial-app/ folder
   - Windows: Shift + Right-click ‚Üí "Open PowerShell window here"
   - Mac: Right-click ‚Üí Services ‚Üí New Terminal at Folder

2. Type this command:
   npm install

3. Wait ~1-2 minutes while it downloads React


STEP 5: START THE APP
----------------------------------
In the same terminal, type:
   npm start

The app will open automatically in your browser at:
   http://localhost:3000

================================================================================
USING THE QUESTIONNAIRE
================================================================================

1. Answer all 9 steps of questions
2. Click "Submit & Match Trials" 
3. Your responses will download as a JSON file
4. You'll see matching results on screen

The questionnaire evaluates:
- 19 clinical trials
- 75 criteria (from 2 clusters: CMB, PTH)

Note: Currently incomplete - only 2 of 9 clusters are in the database.
